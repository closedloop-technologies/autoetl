# autoetl

Wire up APIs with databases in minutes. Ingest, persist, enrich, merge and serve data automagically.


## Project Structure

Creates a new project directory
```
automath init etl_project
tree -av
```

Creates this folder structure
```bash
.
├── .env
├── apis
├── autoetl.yaml
├── db
├── deployment
├── domain_knowledge
├── etl
├── fns
└── serve
```
